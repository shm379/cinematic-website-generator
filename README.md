# 🎬 سازنده‌ی وب‌سایت سینمایی — Cinematic Website Generator

پنلی که با گفتنِ **نام برند** و **حوزه‌ی کاری**، یک **وب‌سایت سینمایی تک‌صفحه‌ای اسکرول‌محور** برایت می‌سازد — هم‌سبکِ نمونه‌ی [«خشخاش»](public/demo/index.html): لودینگ، هیروی پین‌شده، روایت‌های واژه‌به‌واژه، کالکشن محصول/خدمت، خبرنامه و فوتر.

> ساخته‌شده بر پایه‌ی **GSAP + ScrollTrigger + Lenis**. بدون React، بدون TypeScript، بدون Build Step. خروجی هر سایت یک **فایل HTML خودکفا** است.

---

## ✨ این پروژه چه می‌کند؟

نمونه‌ی اصلی (دمنوش «خشخاش») به ۲۲۷ فریمِ استخراج‌شده از ویدیو وابسته بود و فقط برای همان برند کار می‌کرد. این پروژه آن طرح را به یک **موتور پارامتریک** تبدیل کرده:

* **پنل Builder** (`/`): فرم + **پیش‌نمایش زنده** + دکمه‌های **دانلود / کپی کد / باز کردن در تب جدید**.
* **هیروی پروسیجرال با Canvas**: به‌جای فریم‌های ویدیو، یک صحنه‌ی اتمسفریک (نور، ذرات، گرادیان، پارالاکس) که **۱۰۰٪ با اسکرول کنترل می‌شود** و رنگ و موتیفش با هر حوزه تطبیق می‌یابد — پس برای **هر حوزه‌ای** بدون نیاز به ویدیو کار می‌کند.
* **پریست‌های آماده‌ی حوزه‌ها**: چای/دمنوش، قهوه، عطر و زیبایی، طلا و جواهر، املاک، رستوران، باشگاه، کلینیک، فناوری، مد — هرکدام با پالت، موتیف و متنِ پیش‌فرضِ مناسب (همه قابل ویرایش).
* **دو زبانه**: فارسی (RTL، فونت Vazirmatn) و انگلیسی (LTR، فونت Cormorant + Inter).

## 🚀 اجرا

```bash
npm install
node server.js
```

سپس:

| مسیر | توضیح |
|------|-------|
| `http://localhost:3000/` | **پنل Builder** |
| `http://localhost:3000/demo` | نمونه‌ی واقعی و پولیش‌شده («خشخاش») |
| `POST /api/generate` | تولید سمت‌سرور؛ بدنه = config، خروجی = `{ html }` |
| `GET /api/site?field=coffee&brand=My%20Cafe` | رندر مستقیم یک سایت تولیدشده |

اگر پورت ۳۰۰۰ اشغال باشد، سرور خودکار به اولین پورت آزاد می‌رود.

> پنل کاملاً **کلاینت‌ساید** است؛ یعنی روی هاست استاتیک (مثل Vercel) هم بدون نیاز به سرور کار می‌کند. سرور Express فقط برای راحتی اجرای محلی + API اختیاری است.

## 🧩 ساختار

```text
project/
├── server.js                 ← سرور Express (پنل + دمو + API)
├── MASTER-PROMPT.md          ← اسپکِ کاملِ طراحیِ این سبک وب‌سایت‌ها
└── public/
    ├── index.html            ← پنل Builder (صفحه‌ی اصلی)
    ├── generator.js          ← موتور تولید (مرورگر + Node، UMD)
    ├── demo/index.html       ← نمونه‌ی واقعی «خشخاش» (هیروی فریم‌محور)
    ├── frames/               ← ۲۲۷ فریمِ دمو
    └── dawn.jpg / dusk.jpg / night.jpg / newsletter-bg.jpg
```

## 🛠 استفاده‌ی برنامه‌نویسی از موتور

`public/generator.js` یک ماژول UMD است؛ هم در مرورگر (`window.CWG`) و هم در Node کار می‌کند:

```js
const CWG = require('./public/generator.js');

const html = CWG.generate({
  brand: 'خشخاش',
  field: 'tea',            // یکی از کلیدهای CWG.presets یا '_default'
  lang: 'fa',              // 'fa' | 'en'
  accent: '#f59e0b',       // اختیاری؛ پیش‌فرض از روی حوزه
  heroTitle: 'دمنوش',
  overlays: [['یک جرعه.', 'حکمتی کهن درون.'], /* ... */],
  items: [{ name: 'سپیده‌دم', desc: '...', price: '...' }, /* ۳ کارت */]
});
// html = یک فایل کاملِ خودکفا
```

هر فیلدی که ندهی، از پریستِ حوزه پر می‌شود (`CWG.withDefaults`).
